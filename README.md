# CHIP8-C ++ 
CHIP8 emulator in C++

Files Structure:
-> CHIP8_cpu.cpp      Responsible for actually emulating the hardware
-> Graphics.cpp       Uses SDL2 to draw graphics 
-> Input.cpp          Uses SDL2 for handling inputs from the keyboard
-> Audio.cpp          Uses SDL2 for monotone sound. 
-> main.cpp           The conductor that pulls all these elements together to make a working emulator.
